# ScholarBridge 🎓✨

**ScholarBridge** is a premium academic services portfolio and research consulting platform. It connects students and researchers with professional academic support, including project writing, seminar documentation, and research consultation.

![ScholarBridge Banner](public/logo.jpg)

## 🚀 Live Demo
[View Live Site](https://scholarbridge.site)

---

## ✨ Key Features
- **Research Gallery**: A comprehensive, filtered portfolio featuring over 20+ real-world academic projects across various disciplines (Blockchain, AI, Engineering, etc.).
- **Interactive Abstracts**: High-fidelity modal system for viewing research summaries without leaving the page.
- **Smart Lead Generation**: A contextual contact form that automatically pre-selects services based on user navigation.
- **Direct WhatsApp Integration**: Instant "Inquire" buttons that redirect users to a pre-filled WhatsApp chat for faster communication.
- **Responsive Premium Design**: Sleek "Deep Navy & Slate" aesthetic with smooth Framer Motion animations.

## 🛠️ Tech Stack
- **Frontend**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router v6](https://reactrouter.com/)

## 📂 Project Structure
```bash
src/
├── assets/             # Images and brand logos
├── components/         # Reusable UI components (Modal, Navbar, Footer, etc.)
├── constants/          # Centralized data (Portfolio items, Service lists)
├── pages/              # Main view components (Home, Services, Portfolio)
├── utils/              # Utility functions (WhatsApp API helpers)
└── App.jsx             # Main Application Logic
```

## ⚙️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Aja-Cornelius/ScholarBridge.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd ScholarBridge
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```
